# Make a Picture

A CSS art project featuring a nighttime house scene with animated elements. Created using only HTML and CSS.

## Features

- Night sky with moon and twinkling stars
- House with glowing window and chimney
- Animated smoke rising from the chimney
- Framed painting aesthetic with hover effects

## Project Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations

## Usage

Open `index.html` in a web browser to view the scene. No build process or dependencies required.

## Technologies

- HTML5
- CSS3 (animations, gradients, clip-path)
